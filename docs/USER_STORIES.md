> 220 историй по 21 эпику. Формат: **Как** роль, **хочу** функцию, **чтобы** ценность.
> Роли: 👤 владелец · 🛠 оператор · 👨‍💻 разработчик · 🔐 безопасник · 🙋 гость.
>
> **Как фильтровать:** проставь статус в начале строки — `[x]` done · `[~]` partial · `[ ]` todo · `[-]` drop.
> Многое из эпиков «Голос», «Личность», «Проактивность», «Мозг» уже реализовано
> в текущей работе (Qwen3-14B на GPU, Piper, персона, host+cluster проактивность) —
> см. `docs/JARVIS_ALIVE.md` и memory `jarvis-movie-upgrade`.
