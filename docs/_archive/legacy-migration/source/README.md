# Legacy Source Snapshots

This folder keeps the small archive-derived source files that were directly used to build the current migration.

Kept here on purpose:

- `voicecommands.xml`
- `voicecommands.beforeApplyingConfig.xml`
- `command_inventory.csv`
- `command_inventory_full.csv`
- `options.xml`
- `payloads/payloadAllButtonNames.xml`

Not kept here:

- the full extracted legacy dump
- Windows installers and scripts
- logs, DB dumps, plugin caches, and other non-runtime bulk artifacts

Those remain documented in the audit CSV files under the parent directory.
