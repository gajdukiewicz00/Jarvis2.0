# Archived Tool Definitions (Legacy Drafts)

These JSON files are **early drafts** of LLM tool schemas.
They are **not used at runtime** and are kept here for historical reference only.

## Source of Truth

The active tool registry is:

```
apps/llm-service/src/main/resources/tools/registry.json
```

Loaded by `ToolSchemaRegistry.java` in `llm-service`.

Any tool schema changes should be made there, not in these archived files.
